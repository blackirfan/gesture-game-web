# 🎮 Gesture Fighter – Hand Gesture Controlled Web Game

## 🚀 Overview

Gesture Fighter is an interactive **browser-based arcade game controlled using hand gestures**.  
The game combines **Computer Vision**, **Flask**, **MediaPipe**, and **HTML5 Canvas** to create a real-time gesture-controlled gaming experience.

Instead of using a keyboard or controller, the player uses **hand movements detected from a webcam** to control the fighter plane and shoot enemies.

This project demonstrates how **AI-powered perception (computer vision)** can be integrated into **interactive web applications and games**.

🚀 **Built a Hand Gesture Controlled Web Game using Computer Vision!**

I recently developed **Gesture Fighter**, a browser-based arcade game where the player controls a fighter plane using **hand gestures detected from a webcam**.

Instead of traditional keyboard controls, the game uses **MediaPipe hand tracking and OpenCV** to interpret gestures in real time and send them to a **Flask backend**, which drives the game logic rendered in the browser using **HTML5 Canvas and JavaScript**.

### 🎮 Game Features
- ✈️ Fighter plane controlled by hand movement
- 🖐 Gesture-based shooting
- ❤️ Life system
- 📈 Level progression with increasing difficulty
- 🏆 Top 5 leaderboard stored on the server
- 👤 Player name input before starting the game
- 🎯 Real-time gesture detection using a webcam

This project helped me explore how **Computer Vision can be used as an input interface for interactive systems and games**.
